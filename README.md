### TikiTimer
An implementation of the pomodora technique.

### Stack
- HTML
- CSS
- JavaScript
- TypeScript

### How To Run Locally
- Clone the repository:
```
    git clone https://github.com/chyna-gvng/tiki-timer.git
```
- Install dependancies:
```
    npm install
```
-   Run application:
```
    npm run dev
```
